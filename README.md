# Welcome to EmdaDiscoverAshland
The goal of this game is to solve the puzzle to reveal an image. 

To play the game, please select an image adjacent to the empty square.
Then, slide the image tile into the empty spot! 

If you are stuck, hover over the discover ashland for a hint and description of the image! 

If you'd like, there is music available to play on the bottom right of the screen.

Happy discovering!
